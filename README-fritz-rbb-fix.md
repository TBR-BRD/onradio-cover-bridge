# Fritz / rbb 88.8 Fix

Dieser Patch aktualisiert **Radio Fritz** und **rbb 88.8** auf die aktuellen M3U-Playlisten und erweitert den Stream-Resolver um **M3U-Unterstuetzung**. Dadurch koennen beide Sender wieder ueber die aktuelle offizielle Playlist aufgeloest und anschliessend per ICY-Metadaten ausgewertet werden.
